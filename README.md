# SortingScript
Easy sorting script for sort different files in ~/Downloads

how works: 
  check folders in ~/Downloads, if there isnt folders "img, docs" script will create them.
  
  if there are files with images or docs like extentions will move into them.

sorts files: 

img : jpg , jpeg , png , gif 

docs : txt , docx , pdf

scripts : sh , py , bat , cpp
